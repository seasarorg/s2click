ページテンプレート

フォーマットオブジェクト

Clickではページテンプレートにおいて$formatという変数でフォーマットオブジェクトを参照することができます。 フォーマットオブジェクトはHTMLのエスケープやURLエンコードなどを行うユーティリティを提供します。

S2Clickではフォーマットオブジェクトとしてorg.seasar.s2click.util.S2ClickFormatがデフォルトで使用されます。 S2ClickFormatはClick標準のorg.apache.click.util.Formatを拡張し、以下のような便利なメソッドを追加しています。

メソッド 説明
String date(Date date) 引数に渡された日付をyyyy/MM/dd形式にフォーマットします。
String datetime(Date date) 引数に渡された日付をyyyy/MM/dd HH:mm:ss形式にフォーマットします。
String json(Object obj) JavaオブジェクトをJSONに変換します。
String nbsp(String value) 連続する半角スペースの2つめ以降を&nbsp;に変換します。
Object ognl(String expression) 引数のOGNL式として評価し、結果を返却します。
String mask(String value) 引数の文字列と同じ長さのアスタリスクからなる文字列を返却します。

JSPのサポート

Apache Clickでは標準のビュー技術としてVelocityを採用していますが、 Velocity以外にもJSPとFreeMarkerを標準でサポートしています。 S2ClickではJSPの利用を支援するために、フォーマットオブジェクトをEL関数で呼び出す機能を提供しています。

Velocityでは$formatでフォーマットオブジェクトを参照することができます。 フォーマットオブジェクトにはHTMLエスケープやURLエンコーディング、日付の整形など、 ビューを作成する上で便利なメソッドが用意されています。

ClickでJSPを利用する場合、フォーマットオブジェクトはHttpServletRequestに"format"という名前でセットされているので、 JSPでは以下のようなコードで呼び出すことが可能です。 

<%
  // フォーマットオブジェクトを取得
  S2ClickFormat format = (S2ClickFormat) request.getAttribute("format");
%>
<%= format.html(value) %>

S2Clickではスクリプトレットを使用せずにフォーマットオブジェクトを呼び出すことができるよう、 JSP-ELでフォーマットオブジェクトのメソッドを呼び出すための関数を提供します。 先ほどの例は以下のようにELでスマートに記述することができます。 

<%@ taglib uri="http://s2click.sandbox.seasar.org/format" prefix="format" %>
...
${format:html(value)}

S2Clickが提供するEL関数はフォーマットオブジェクトのメソッドと名称が異なるものが一部存在します。 詳細についてはS2ClickFunctionのJavadoc、 もしくはformat.tldを参照してください。
